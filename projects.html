Plane Ticket Management System

This project showcases skills in Java programming, object-oriented principles, and database connectivity, creating a streamlined and functional application for managing airline tickets efficiently.

E-Commerce Store

Create an e-commerce store with a product catalog, search and filter functionality, shopping cart, and secure checkout process. You can integrate a payment gateway for transactions. This project will showcase your full-stack skills, including frontend development, backend APIs, and user session management.

Build a weather dashboard that displays real-time weather data for searched locations. Using a weather API like OpenWeatherMap, allow users to check weather forecasts based on location.
